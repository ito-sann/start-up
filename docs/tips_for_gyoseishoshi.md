# 行政書士向けイベント活用Tips

## 「補助金の人」として認識されるための最短コミュニケーション術

### 🎯 自己紹介テンプレート

> 「行政書士の○○です。**スタートアップ向けの補助金申請を専門**にしています。
> 創業助成金やものづくり補助金など、**採択率80%以上**の実績があります。」

**ポイント:**
- 具体的な数字（採択率）を入れる
- 「スタートアップ専門」という差別化
- 30秒以内で言い切る

---

## 🎪 イベント別アプローチ戦略

### 🔥 ピッチイベント（スコア: 100点）

**最も価値が高いイベント**

| タイミング | アクション |
|-----------|-----------|
| ピッチ前 | 登壇者リストを事前チェック、SNSで事前にフォロー |
| ピッチ中 | メモを取る、質問を考える |
| ピッチ後 | 登壇者に直接声かけ「素晴らしいプレゼンでした！」 |
| 懇親会 | 審査員・投資家と同じタイミングで名刺交換 |

**キラーフレーズ:**
> 「VCからの資金調達と並行して、補助金という選択肢も検討されていますか？
> 返済不要で、事業のエクイティを希薄化させずに資金を得られますよ」

---

### 🤝 交流会・ネットワーキング（スコア: 90点）

**会話の流れ:**

1. **アイスブレイク**: 「何のビジネスをされていますか？」
2. **掘り下げ**: 「どんな課題がありますか？」
3. **提案**: 「その事業なら○○補助金が使えるかもしれません」
4. **クロージング**: 「詳しくお話しませんか？名刺交換させてください」

**NG行動:**
- ❌ いきなり営業トークを始める
- ❌ 名刺を配りまくる
- ❌ 自分の話ばかりする

---

### 🛠️ ワークショップ（スコア: 70点）

- グループワーク中にさりげなく専門性をアピール
- 「資金面で悩んでいる方がいたら、補助金のことなら相談に乗りますよ」
- 終了後に個別で声かけ

---

## 📋 持ち物チェックリスト

- [ ] **名刺（50枚以上）**
  - 裏面に「スタートアップ支援専門」と記載
  - QRコードでLINE公式アカウントへ誘導
  
- [ ] **採択実績シート（A4 1枚）**
  - 直近5件の採択事例
  - 業種・補助金額・採択年月を記載
  
- [ ] **補助金カレンダー（A4 1枚）**
  - 次回締切の補助金一覧
  - 「今申請すれば○月に受給」と分かりやすく
  
- [ ] **スマートフォン**
  - 予約カレンダーをすぐ見せられるように
  - 事例紹介ページをブックマーク

## 🎯 狙うべき起業家の特徴

1. **資金調達を検討中** 
   - VCへのピッチ準備をしている
   - → 事業再構築補助金、ものづくり補助金

2. **新製品・サービス開発中**
   - 開発費用がかかるフェーズ
   - → ものづくり補助金、事業再構築補助金

3. **海外展開を検討中**
   - 輸出・越境EC・海外拠点設立
   - → 海外展開・事業再編資金

4. **IT投資を検討中**
   - システム導入、DX推進
   - → IT導入補助金

5. **設備投資を検討中**
   - 製造設備、店舗改装
   - → ものづくり補助金、小規模事業者持続化補助金

## 📆 イベント参加の黄金ルール

1. **月に最低3回はイベントに参加する**
2. **同じ施設に3回通って顔を覚えてもらう**
3. **名刺交換した人には24時間以内にお礼メール**
4. **SNSでイベント参加報告を投稿（タグ付け）**
5. **3ヶ月後にフォローアップ連絡**
